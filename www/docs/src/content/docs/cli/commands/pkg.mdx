Usage:

<Code
  code="$ nrz pkg [<command>] [<args>]"
  title="Terminal"
  lang="bash"
/>

Get or manipulate package.json values

## Subcommands

### get

Get a single value

<Code code="$ nrz pkg get [<key>]" title="Terminal" lang="bash" />

### init

Initialize a new package.json file in the current directory

<Code code="$ nrz pkg init" title="Terminal" lang="bash" />

### pick

Get multiple values or the entire package

<Code
  code="$ nrz pkg pick [<key> [<key> ...]]"
  title="Terminal"
  lang="bash"
/>

### set

Set one or more key value pairs

<Code
  code="$ nrz pkg set <key>=<value> [<key>=<value> ...]"
  title="Terminal"
  lang="bash"
/>

### delete

Delete one or more keys from the package

<Code
  code="$ nrz pkg delete <key> [<key> ...]"
  title="Terminal"
  lang="bash"
/>

## Examples

Set a value on an object inside an array

<Code
  code={`$ nrz pkg set "array[1].key=value"`}
  title="Terminal"
  lang="bash"
/>

Append a value to an array

<Code
  code={`$ nrz pkg set "array[]=value"`}
  title="Terminal"
  lang="bash"
/>
